Github for  Collaboration

To allow other developers to collaborate on your project, navigate to your project in GitHub, then:

  • Go to the Settings tab (right-most button along the top, sometimes hidden in ... )
    • You'll likely be prompted for your password at this stage
  • Choose Collaborators from the Left Nav sidebar
  • Click Add People
    • Search by username, full name, or email address
  • Once people have been added to your project as collaborators, they can clone, edit, pull, push, and issue pull requests.
