Checking the Status of a Git Repo

The first git command to know is:

  • git status

Running git status will scan your project for new and/or edited files.

  • It will find and list any untracked files that have been created within your project directory or sub-directories.  These will need to be added (next slide) to the repository in order to be under version control.
  • It will also find and list any previously added files in your repository that have been edited since the last commit.
