Clone a Github Repo

Cloning a GitHub repo means to make a local version, including all version history, of a repo hosted on GitHub.

To do so:

  • Log in to GitHub and navigate to the repo that you want to work on locally
  • Click the Clone or Download button
  • Click the Copy to Clipboard button located beside the path to your GitHub repo
  • Then, on your local machine open Terminal and change to the directory where you would like to store your project
  • Enter the following command in your terminal (the path portion is what you copied to clipboard):
    • git clone [path-to-your-github-repo-git-file]
