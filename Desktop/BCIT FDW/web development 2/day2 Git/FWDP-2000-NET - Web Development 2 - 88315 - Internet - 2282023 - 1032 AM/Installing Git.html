Installing Git

BCIT computers for this class already have Git installed, but since they get wiped each evening you'll need to a bit of extra setup each day before you are up and running.

For Windows users you can download and install Git from:

  • https://gitforwindows.org/

For Mac users you can download and install from:

  • http://git-scm.com/download/mac

For either platform, you can download and install the Github for Desktop software, which is a free GUI for managing Git.  Installing Github desktop also installs Git, so for your own laptops it is probably the easiest way to go.

  • https://desktop.github.com/

Note:  GitHub is fundamentally an online portal for managing Git projects and repositories.  Git is the version control software itself.  You can manage projects and repositories without GitHub, but not the other way around.
