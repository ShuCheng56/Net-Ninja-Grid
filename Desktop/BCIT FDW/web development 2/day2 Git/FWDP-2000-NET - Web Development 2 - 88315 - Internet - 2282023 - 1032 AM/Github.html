GitHub

Git and GitHub are two separate things.

Git is a version control system (software).

GitHub is a code hosting service (owned by Microsoft) that stores git repositories in the cloud.  Having your repositories stored in the cloud is beneficial because:

  • It provides redundancy if you lose the local version of your project
  • Provides access to your project from multiple machines
  • Allows multiple developers to easily work on the same project at the same time

GitHub offers both free and paid accounts.  

Free accounts allow you to:

  • Host unlimited public and private repositories
  • Public repositories can be viewed (but not edited) by anyone with a browser and an internet connection.
    • To edit a public repository, you must be the project owner or have invited another GitHub user(s) as a collaborator of the project
    • Public repositories are fine for open-source projects and class projects for education, but may not be appropriate for client work
  • Private repositories can only be viewed and edited by your team.

For this program, a free account is all you need.
