Git Setup

Git requires very little setup to start using once installed.  

The following two global commands only need to be set once per computer after which they will apply to all future projects on that computer.

Git Global Settings: user.name

Sets the name that git will attach to your commits (we'll discuss commits shortly).

Open your Terminal or Command prompt and at the prompt enter the following (replacing with your own name, of course).

  • git config --global user.name "Josh Solomon" 

Git Global Settings: user.email

Sets the email address that will be attached to your commits.

  • git config --global user.email "jsolomon11@bcit.ca" 

Note: As mentioned, because lab computers get scrubbed each night, you will need to re-enter the above two commands each class if you're using a lab computer.
