Track a Remote Repo 

If you already have a git repo for your project on your local machine and you want to push it out to GitHub:

  • Create a repository for your project on GitHub
    • To avoid errors, make sure the options for adding a README, .gitignore, and license are NOT checked.  You want an empty repo on GitHub.
  • At the top of the page on your newly created GitHub repo, click the Copy To Clipboard button next to the repo URL
  • Back on your local computer, in your terminal, change directory to the root of your project
  • Enter the following set of commands:
    • git remote add origin [REMOTE_URL]
      • replace REMOTE_URL with the URL you copied in the previous step
    • git remote -v
      • verify the remote url
    • git push -u origin main
      • this sends your local repository, including your history of commits, to GitHub.

When that's all done, you should see your project's files and revisions on GitHub
