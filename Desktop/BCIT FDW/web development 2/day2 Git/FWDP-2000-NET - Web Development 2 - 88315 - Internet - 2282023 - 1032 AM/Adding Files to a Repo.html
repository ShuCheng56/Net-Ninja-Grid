Adding Files to a Repo

Running git status will notify you of any untracked files that exist in your project that are not yet in the repository.

To add an individual file to your repo, run the following command in your terminal:

  • git add [the path to your file]

To add all untracked files to your repo, git offers a shortcut:

  • git add .

Be aware that for larger projects it is often better to add specific files instead of using the above shortcut, mainly because you want to be specific about what gets added to each commit.
