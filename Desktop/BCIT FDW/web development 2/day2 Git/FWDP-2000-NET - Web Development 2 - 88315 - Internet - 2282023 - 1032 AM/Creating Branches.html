Creating Branches

The default, "base", branch of a git repository is named main.  

You always create a new branch from an existing branch, typically from the  main branch.

View all existing branches

To view all existing branches in a repository, use the following command:

  • git branch -a

The above will also show you what branch you are currently working on.

Switch to a different existing branch

To switch to a different branch, first commit any changes in your current working branch (or risk losing them), then:

  • git checkout [name-of-branch-to-switch-to]

Note: There is a relatively new command for switching branches, but it is still in "experimental" status (despite being stable for a while now).  The new alternative to the above is: git switch [name-of-branch-to-switch-to]         

Create a new branch

To create a new branch based on the current branch, enter:

  • git branch [your-new-branch-name]

Note:  We can also use the switch command, with a -C flag, to create and switch to a new branch in one step.  git switch -C [your-new-branch-name]

Upload current branch to GitHub

  • git push –u origin [your-current-branch-name]
