Git - About

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

It was created by Linus Torvalds in 2005 for use in the development of Linux. 

http://git-scm.com/

Git can be used via:

  • The command line - entering text commands via the terminal (Mac) or Command Prompt (windows) 
  • through a GUI

Though we'll take a look at both, I would encourage you to get comfortable with at least the basics of using git from the command line as that will often be the only choice when working on remote systems (via ssh).
